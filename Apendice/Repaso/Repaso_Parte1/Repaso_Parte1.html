Sistemas de ecuaciones lineales¶

La librera numpy.linalg tiene diversas herramientas para resolver y caracterizar sistemas de ecuaciones en python.

Consideremos un sistema de ecuaciones de la forma

\begin{eqnarray*} 3x_1 + 1x_2 - 5x_3 &=& 2 \\ -2x_1 - 2x_2 + 5x_3 &=& 5 \\ 8x_1 + 1\times10^{10}x_2 + 2x_3 &=& -3 \\ \end{eqnarray*}

Solución del sistema (numpy.linalg.solve)¶

Para resolver esto en python, primero debemos escribir el sistema en forma matricial:

In [1]:
import numpy as np
A = np.array([[ 3,  1, -5],
              [-2, -2,  5],
              [ 8,  1E10,  2]])

y = np.array([[2], 
              [5], 
              [-3]])

La solucin a este sistema se obtiene mediante numpy.linalg.solve

In [2]:
import numpy as np
xsol = np.linalg.solve(A,y)
print('La solucin es x =\n',xsol)

La solucin es x =
 [[ 6.99999999e+00]
 [-6.65999999e-09]
 [ 3.79999999e+00]]

Condicionamiento de la matriz (numpy.linalg.cond)¶

Una forma de caracterizar el sistema es mediante el nmero de condicin. En python numpy.linalg.cond

In [3]:
print('Nmero de condicionamiento, cond(A) =', np.linalg.cond(A))

Nmero de condicionamiento, cond(A) = 15824113618.647936

El nmero de condicin est directamente relacionado con el determinante de la matriz. Si $\mathrm{cond}(A)$ es muy grande, entonces $\mathrm{det}(A)\approx 0$. As:

A mayor nmero de condicionamiento, mayor es la probabilidad de inducir errores de redondeo, overflow o underflow en la solucin.

Cuantificación del error (numpy.linalg.norm)¶

Podemos revisar el error mediante la norma Frobenius. En python, usamos numpy.linalg.norm

En este ejemplo, comprobamos el error de la solcin con $\|Ax - y\|$

In [4]:
print('Error |A*xsol - y|', np.linalg.norm(np.dot(A,xsol) - y))

Error |A*xsol - y| 5.329070518200751e-15

Número de soluciones posibles (numpy.linalg.matrix_rank)¶

Para analizar si el sistema tiene solucin nica, infintas soluciones o no tiene solucin, evaluamos los $\mathrm{rank}(A)$ y $\mathrm{rank}([A|y])$, y el nmero de incognitas $n$.

En python usamos numpy.linalg.matrix_rank para determinar el rango, y len(y) para determinar $n$.

In [5]:
from numpy.linalg import matrix_rank
Ay = np.concatenate((A,y),axis=1) # generamos matrix aumentada
rankA  = matrix_rank(A)           # rango de A
rankAy = matrix_rank(Ay)          # rago de A|y
n = len(y)                        # nmero de incognitas

if rankAy == rankA:
    if rankA == n: print('El sistema tiene solucin nica')
    if rankA < n: print('El sistema tiene infinitas soluciones')
elif rankAy == rankA + 1: print('El sistema no tiene soluciones')

El sistema tiene solucin nica

Regresión lineal (numpy.linalg.pinv o .lstsq)¶

Una manera de obtener $\alpha$, $\beta$ y $\rho$ es medinte regresin lineal a partir de un modelo linealizado.

paso 1. Linealizamos el modelo: \begin{equation*} U(v,t) = \alpha \frac{(v + 2)^{\beta}}{(t + 3)^{3\rho}} \Rightarrow \ln(U) = \ln(\alpha) + \beta\ln(v + 2) - \rho~ \ln(t+3)^3 \end{equation*}

Obtenemos un modelo lineal 2D de la forma $y = a_0 + a_1x_1 + a_2x_2$, con:

  • $y = \ln(U)$
  • $a_0 = \ln(\alpha)$
  • $a_1 = \beta$
  • $x_1 = \ln(v + 2)$
  • $a_2 = -\rho$
  • $x_2 = \ln(t+3)^3$

paso 2. Transformamos los datos $x_i$, $y_i$ y $z_i$ al modelo linealizado

In [7]:
x1i = np.log(vi + 2)
x2i = np.log((ti+3)**3)
yi  = np.log(Ui)

paso 3. A partir del modelo general $y = a_0z_0+a_1z_1+a_2z_2+...a_nz_n$, creamos la matriz:

\begin{equation*} Z =\begin{bmatrix} 1 & x_{1,1} & x_{2,1}\\ 1 & x_{1,2} & x_{2,2}\\ \vdots & \vdots & \vdots \\ 1 & x_{1,m} & x_{2,m} \end{bmatrix} \end{equation*}
In [8]:
Z = np.vstack((x1i**0, x1i**1, x2i**1)).T

paso 4. Hacemos ajuste por mnimos cuadrados. En python, a travs de la librera numpy.linalg, tenemos dos alternativas:

  • Mtodo matriz psudoinversa, $Z_\mathrm{pinv} = \left(Z^T Z\right)^{-1}Z^T$ (pinv)
    Z_pinv = np.linalg.pinv(Z) # Matrz Z pseudo inversa
a = np.dot(Z_pinv,yi)      # Parmetros del modelo lineal
  • Mnimos cuadrados (lstsq)
    a = np.linalg.lstsq(Z, yi, rcond=None)[0] # Parmetros del modelo lineal
In [9]:
a = np.dot(np.linalg.pinv(Z),yi)
print('modelo lineal, y = %.3f + %.3fx_1 + %.3fx_2' % tuple(a))

modelo lineal, y = -0.959 + 1.973x_1 + -1.184x_2

paso 5 Extraemos los parmetros del modelo original. En este caso:

  • $\alpha = e^{a_0}$
  • $\beta = a_1$
  • $\rho = a_2$
In [10]:
alpha = np.exp(a[0])
beta  = a[1]
rho   = - a[2]
print('alpha = %.3f, beta = %.3f, gamma = %.3f' % (alpha, beta, rho))

alpha = 0.383, beta = 1.973, gamma = 1.184

Regresión no lineal (scipy.optimize.curve_fit)¶

En el caso de regresin no lineal, el proceso es ms sencicllo. En python, ocupamos la funcin curve_fit de la librera scipy.optimize.

from scipy.optimize import curve_fit

def model(x,a,b,c):
    U = a*(x[0] + 2)**b/(x[1] + 3)**(3*c)
    return U
    
xdata = np.vstack((vi,ti))

b = curve_fit(model,xdata,ydata)[0]
print('alpha = %.3f, beta = %.3f, gamma = %.3f' % tuple(b))

Calidad del modelo (sklearn.metrics.r2_score)¶

Evaluamos la calidad del modelo mediante el coeficiente de determinacin. En python usamos la funcin r2_score de sklearn.metrics.

Primero creamos una funcin para nuestro modelo. En este caso crearemos dos funciones:

  • U_lin para el modelo con los prametros obtenidos por regresin lineal
  • U_nl para el modelo con los prametros obtenidos por regresin lineal
In [12]:
U_lin = lambda v,t: alpha*(v+2)**beta/(t + 3)**(3*rho) # regresin lineal
U_nl  = lambda v,t: b[0]*(v+2)**b[1]/(t + 3)**(3*b[2]) # regresin no-lineal
In [13]:
from sklearn.metrics import r2_score
print('r2 (reg. lineal) = ',    r2_score(U_lin(vi,ti), Ui))
print('r2 (reg. no-lineal) = ', r2_score(U_nl(vi,ti) , Ui))

r2 (reg. lineal) =  0.9964786187961536
r2 (reg. no-lineal) =  0.9989416972396143

Interpolación polinomimal¶

En python tenemos distintos mtodos para generar polinomios de interpolacin. En el curso vimos 3 mtodos:

  • Usando mnimos cuadrados, mediante la librera numpy:

    import numpy as np
ai = np.polyfit(xi,yi,len(xi) - 1) # generamos los coeficientes del polinomio
y0 = np.polyval(ai, x0)            # evaluamos el polinomio en x0 para encontrar y0

  • Polinomios de Newton, mediante el script newton.py desarrollado en clases:

    import newton as nw
ai = nw.divided_diff(xi,yi)       # generamos los coeficientes del polinomio de Newton
y0 = nw.newnton_poly(ai, xi, x0)  # evaluamos el polinomio en x0 para encontrar y0
  • Polinomios de Lagrange, mediante la librera scipy.interpolate.
    from scipy.interpolate import lagrange
y_interpol = lagrange(xi,yi)    # generamos una funcin de interpolacin
y0 = y_interpol(x0)             # evaluamos el polinomio en x0 para encontrar y0

Notar que tanto el mtodo de regresin lineal como polinomios de newton se basan en una funcin para generar los coeficientes (polyfit y divided_diff, respectivamente), y luego otra funcin para evaluar el polinomio de interpolacin (polyval y newton_poly, respectivamente)

El mtodo de polinomios de Lagrange, en cambio, se basa en la funcin lagrange para generar una funcin de interpolacin (output del tipo "callable" en lenguage python).

Generación de polinomios¶

En el siguiente script, utilizamos las tres funciones para generar una funcin de interpolacin. Notar que usamos funcin lambda para regresin lineal y polinomios de newton:

In [15]:
import newton as nw # funciones del script newton.py
from scipy.interpolate import lagrange

# regresin lineal
a_lr = np.polyfit(xi,yi,len(xi)-1)              # generamos los coeficientes del polinomio
y_linreg   = lambda x: np.polyval(a_lr,x)       # creamos un output tipo "callable" a travs de funcin lambda

# polinomios de Newton
a_nw = nw.divided_diff(xi,yi)                   # generamos los coeficientes del polinomio
y_newton = lambda x: nw.newton_poly(a_nw,xi, x) # creamos un output tipo "callable" a travs de funcin lambda

# polinomios de lagrange   
y_lagrange = lagrange(xi,yi)                     # creamos un output tipo "callable"

Extrapolación en interp1d¶

Es importante mencionar que, por defecto, interp1d arroja un error para extrapolacion.

Por ejemplo, en nuestro caso min(xi) = -2.9924. As, si evaluamos el polinomio en x0 = -3 veremos un error:

Condiciones de borde en CubicSpline¶

CubicSpline permite especificar las condiciones de borde mediante bc_type. Para funciones no peridicas, tenemos:

from scipy.interpolate import CubicSpline
y = CubicSpline(xi,yi,bc_type='not-a-knot') # Condicin de borde not a knot

y = CubicSpline(xi,yi,bc_type='clamped')    # Condicin de borde "clamped"

y = CubicSpline(xi,yi,bc_type='natural')    # Condicin de borde"natural"

CubicSpline, por defecto, permite extrapolar.

Generemos tres tipos de interpolacin spline cbica con extrapolacin:

In [19]:
from scipy.interpolate import CubicSpline
y_scubic_nk = CubicSpline(xi,yi,bc_type='not-a-knot')
y_scubic_nat = CubicSpline(xi,yi,bc_type='natural')
y_scubic_cmp = CubicSpline(xi,yi,bc_type='clamped')
